# Wisland Repo
<hr/>
This is a repo for all notes and work related to the world of Wisland. The people, the language, story ideas, notes, concepts etc. 

This is mostly for my own use, and to work on things as I have time and from multiple computers. Although nothing in this repo or any linked ones are *private* per se, this is all my own work. I ask that you be respectful of that. 

While I don't anticipate any PRs, if for some reason you take interest in anything here, or are curious. Please reach out, I would love to hear about it. (PRs for typos or something are fine and appreciated)

No AI was or will be used for any part of this, and any I ask that any PRs do not use AI. I also do not consent for anything here to be used for AI training. (although I'm sure it has been scraped anyway)

Needless to say, this is not a codebase. It is a glorified notebook. Don't expect any build instructions or long term support. :p
