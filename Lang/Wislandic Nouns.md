
There are 5 main classes or declensions of noun in Wislandic.

## N-Open

|      | Singular | Plural |
| :--: | :------: | :----: |
| Nom. |    -     |  -lh   |
| Gen. |    -n    |  -nne  |
| Acc. |   -th    |  -the  |
| Dat. |    -s    |  -sh   |

## N-Closed

|      | Singular |  Plural  |
| :--: | :------: | :------: |
| Nom. |  -C[^1]  | -CC[^1]e |
| Gen. |   -an    |   -en    |
| Acc. |   -ath   |   -the   |
| Dat. |  -(a)s   |  -(a)sh  |

## M

|      | Singular | Plural |
| :--: | :------: | :----: |
| Nom. |    -m    |  -mme  |
| Gen. |    -g    |  -gur  |
| Acc. |   -ps    |  -pse  |
| Dat. |   -ks    |  -kse  |

## Th

|      | Singular |  Plural   |
| :--: | :------: | :-------: |
| Nom. |   -th    | -V[^2]den |
| Gen. |  -thes   |   -thre   |
| Acc. |   -th    | -tlV[^2]  |
| Dat. |  -thesh  |  -thres   |

## Ur
|      | Singular | Plural |
| :--: | :------: | :----: |
| Nom. |   -ur    |  -ar   |
| Gen. |   -urn   |  -arn  |
| Acc. |  -urth   | -arth  |
| Dat. |   -urs   |  -ars  |

[^1]: C here denotes any consonat that is not *m*, *th*, or *ur*. The final consonat of the nominative singular

[^2]: V here denotes the last vowel that precedes the *th* in the nominative singular
