
There are 5 main classes or declensions of noun in Wislandic.

## N-Open

|      | Singular | Plural |
| :--: | :------: | :----: |
| Nom. |    -     |  -lh   |
| Gen. |    -n    |  -nne  |
| Acc. |   -th    |  -the  |
| Dat. |    -s    |  -sh   |
## Ex: Ika *Magic*
|      | Singular | Plural |
| :--: | :------: | :----: |
| Nom. |   Ika    | Ikalh  |
| Gen. |   Ikan   | Ikanne |
| Acc. |  Ikath   | Ikathe |
| Dat. |   Ikas   | Ikash  |
